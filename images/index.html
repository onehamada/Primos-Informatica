Primos-Informatica

Pasta de Imagens

Esta pasta contém as imagens otimizadas para o site da Primos Informática.

Estrutura de Pastas

  • /products/ - Imagens dos produtos
  • /icons/ - Ícones e logos
  • /banners/ - Banners promocionais

Diretrizes de Imagens

  • Formato preferencial: WebP (com fallback para JPG/PNG)
  • Largura máxima: 800px
  • Qualidade: 80% para JPG/WebP
  • Nomes descritivos em minúsculas com hífens (ex: fone-jbl-t520bt.jpg)

Ferramentas Recomendadas

  • Squoosh - Para otimização de imagens
  • TinyPNG - Para compressão adicional

Tamanhos Recomendados

  • Miniaturas de produtos: 300x300px
  • Imagens de produto: 800x800px
  • Banners: 1200x400px

Atualização

Sempre otimize as imagens antes de fazer upload para garantir o melhor desempenho do site.
