# Minha Loja

Este projeto é uma loja online simples, desenvolvida em HTML, CSS e JavaScript. Abaixo estão as instruções para configurar e executar o projeto, bem como detalhes sobre suas funcionalidades.
Acesse AQUI: https://onehamada.github.io/Primos-Informatica/

## Estrutura do Projeto

```
minha-loja
├── src
│   ├── index.html        # Estrutura HTML da loja
│   ├── css
│   │   └── styles.css    # Estilos CSS da loja
│   └── js
│       └── script.js     # Scripts JavaScript para interatividade
├── assets
│   └── images            # Imagens dos produtos
└── README.md             # Documentação do projeto
```

## Como Configurar

1. **Clone o repositório:**
   ```bash
   git clone <URL_DO_REPOSITORIO>
   cd minha-loja
   ```

2. **Abra o arquivo `src/index.html` em um navegador:**
   - Você pode abrir diretamente o arquivo HTML em um navegador ou usar um servidor local para visualizar o projeto.

## Funcionalidades

- **Busca de Produtos:** O projeto inclui um campo de busca para filtrar produtos.
- **Categorias de Produtos:** Os produtos estão organizados em categorias para facilitar a navegação.
- **Adição ao Carrinho:** Os usuários podem adicionar produtos ao carrinho de compras.
